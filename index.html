How to play

Here is how to play Wordle:

  1. Guess a 5-letter word.
    Type and submit your first guess.
  2. Check the colors:
    • Green: Correct letter in the correct spot.
    • Yellow: Correct letter in the wrong spot.
    • Gray: Letter not in the word.
  3. Guess again.
    Use the color clues to try a new word.
  4. You have 6 tries.
    Find the correct word before you run out!

Game Features

  • Random 5-letter word selected from a large dictionary.
  • Color-coded feedback (green, yellow, gray) per letter.
  • Modal popup with game instructions.
